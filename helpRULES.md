# 프로젝트 명: Campus Club AI Interviewer (동아리 면접 도우미)

## 1. 프로젝트 개요
- 대학 동아리 운영진이 지원서를 입력하면 AI가 핵심 역량을 분석하고 날카로운 면접 질문을 생성해주는 서비스.
- 9시간 안에 완성 및 배포해야 하는 1인 해커톤 프로젝트임.

## 2. 기술 스택 (기술적 결정)
- Frontend: React (Vite), Tailwind CSS
- Deployment: Vercel CLI
- AI 모델: gemini-2.5-pro / sonnet 4.6
- UI 라이브러리: Lucide React (아이콘용), Headless UI (필요 시)

## 3. 개발 원칙 (중요)
- **단일 페이지 원칙 (SPA):** 복잡한 라우팅 없이 `App.jsx` 중심의 단일 페이지로 모든 기능을 구현한다.
- **데이터 저장 지양:** 별도의 DB(Firebase 등) 없이, 모든 데이터는 메모리나 LocalStorage에서 처리한다.
- **UI 스타일:** 깔끔한 B2B SaaS 느낌을 위해 흰색 배경에 Indigo 혹은 Blue 포인트를 사용한다.
- **에러 핸들링:** 에러가 발생하면 사용자에게 텍스트로 명확히 알리고, 모든 에러 로그는 콘솔에 출력하도록 짠다.

## 4. 기능 명세
1. 입력창: 지원서 본문을 통째로 붙여넣을 수 있는 대형 Textarea.
2. 분석 버튼: 클릭 시 AI가 분석을 시작하며 로딩 애니메이션 표시.
3. 결과 창: 마크다운 포맷으로 출력 (3줄 요약, 핵심 키워드, 추천 면접 질문 5가지).
4. 복사 기능: 결과물을 클릭 한 번으로 복사할 수 있는 기능.

## 5. 프롬프트 가이드라인 (AI 에이전트용)
- 면접 질문은 지원서의 '모순점'을 찾아내는 날카로운 질문 2개와 '역량 검증' 질문 3개로 구성할 것.
- 어투는 전문적이고 시니어 인사 담당자 같은 느낌을 유지할 것.
